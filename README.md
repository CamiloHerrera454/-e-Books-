## Objetivo del programa
Desarrollar un aplicativo que expone servicios web (API REST) para [describir en 1 línea el objetivo], usando serialización JSON para el intercambio de datos entre cliente y servidor.

## Funcionalidades principales
El sistema incluye al menos 8 servicios web con diferentes funcionalidades:

1. **GET /api/status** → Verifica el estado del servicio.
2. **POST /api/login** → Autenticación de usuario (simulada o real).
3. **GET /api/users** → Lista usuarios (con paginación si aplica).
4. **GET /api/users/{id}** → Consulta un usuario específico.
5. **POST /api/users** → Registra un usuario.
6. **PUT /api/users/{id}** → Actualiza datos de usuario.
7. **DELETE /api/users/{id}** → Elimina usuario.
8. **GET /api/reports** → Genera un reporte/resumen (ej: estadísticas del sistema).
(+ opcionales: /api/search, /api/notifications, /api/history, etc.)

> Nota: Todos los endpoints trabajan con **JSON** para request/response.
